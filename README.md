# Jihong AI System

這是一套提供給任何 AI 使用的個人化 context 系統。

目標：

- 快速讓 AI 理解我是誰
- 成為長期 AI Agent
- 協助寫作、職涯、技術成長
- 可透過 Git 持續版本管理

## 使用方式

新 AI 對話時，先提供以下順序：

1. 00_core/CORE_CONTEXT.md
2. 02_writing/WRITING_STYLE.md
3. 當前任務對應 agent 檔案

例如：

寫文章：
04_agents/ARTICLE_WRITER.md

翻譯：
04_agents/TRANSLATOR.md

職涯規劃：
04_agents/CAREER_MENTOR.md
